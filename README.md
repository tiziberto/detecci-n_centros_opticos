# deteccion_centros_opticos

Este código implementa un sistema diseñado para detectar y rastrear con precisión los centroides de múltiples LEDs en un flujo de video.

El objetivo principal de este código es lograr una detección robusta y estable, incluso cuando los LEDs parpadean de forma inconsistente o son momentáneamente oscurecidos (oclusión). Para ello, emplea una estrategia híbrida que combina dos métodos principales de detección y añade una capa de memoria persistente.
